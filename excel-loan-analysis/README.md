## Loan Portfolio Analysis (Excel)

### Objective
Analyze loan application data to understand approval trends and basic risk segmentation relevant to banking operations.

### Tools Used
Microsoft Excel

### Dataset
Public loan application dataset used for learning and analysis purposes.

### What I Did
- Reviewed loan application data and checked for missing or inconsistent values
- Built PivotTables to summarize loan approval outcomes
- Segmented applicants by income level to compare approval trends
- Created charts to visualize key findings
- Summarized insights in plain language for business review

### Key Insights
- Majority of loan applications were approved in the dataset
- Higher income groups showed more consistent approval outcomes
- Loan amounts varied across applicants, indicating different risk profiles
